Detailed Description

template<typename T>
class LV::IntrusivePtr< T >

Intrusive smart pointer class template.

Template Parameters
TClass to reference count
Note
This is an implementation of boost::intrusive_ptr.

The type T must have two functions overloaded:

  • void intrusive_ptr_add_ref(T* object) – Called to add a reference
  • void intrusive_ptr_release(T* object) – Called to remove a reference and destroy the object when not longer used
