We are back!

After a 5-year hiatus, we are back! Progress is moving swiftly on the next stable release 0.5.0. Among the changes already made are, a new CMake build system, Android support, and a new commandline tool for running visualizations. We also have a new Wiki here.

Working with LV

We are now on GitHub and use Git for versioning.

git clone https://github.com/Libvisual/libvisual.git cd libvisual; cmake . && make install

About LV

Libvisual instantly transforms your favourite music into psychedelic art. It works by running audio streams through an impressive array of effects that crunches the waveform into pixels. With the right drivers, you can display visuals on just about anything.

Use LV

Get a taste of Libvisual by trying out the latest development snapshot off our GitHub page. Even though we are still far from the next official release, this should be quite stable and good for general use. Instructions on building and installation can be found in the user guide.

Develop LV

We need all the help we can get to bring Libvisual to the next level. Areas that need always work are coding, documentation, translations, and testing. Send us an email at our mailing list, or drop by our IRC channel on FreeNode #libvisual.
