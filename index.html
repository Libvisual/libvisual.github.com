We are back!

After a 5-year hiatus, we are back! Progress is moving swiftly on the next stable release 0.5.0. Among the changes already made are, a new CMake build system, Android support, and a new tool for testing and benchmarking.

Working with LV

We are now on GitHub and use Git for versioning.

git clone https://github.com/Libvisual/libvisual.git cd libvisual; cmake . && make install

About LV

Libvisual instantly transforms your favourite music into psychedelic art. It works by running audio from your microphones and speakers through an impressive array of effects that crunches the waveform into pixels. With the right drivers, you can display visuals on just about anything.
