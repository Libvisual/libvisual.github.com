Function Documentation

int visual_audio_analyze ( VisAudio *  audio)

This function analyzes the VisAudio, the Fourier frequency magic gets done here, also the audio energy is calculated and some other magic to provide the developer more information about the current sample and the stream.

For every sample that is being retrieved this needs to be called. However keep in mind that the VisBin runs it automaticly.

Parameters:
audioPointer to a VisAudio that needs to be analyzed.
Returns:
VISUAL_OK on success, -VISUAL_ERROR_AUDIO_NULL on failure.

int visual_audio_init ( VisAudio *  audio)

Initializes a VisAudio, this should not be used to reset a VisAudio.

The resulting initialized VisAudio is a valid VisObject even if it was not allocated. Keep in mind that VisAudio structures that were created by visual_audio_new() should not be passed to visual_audio_init().

See also:
visual_audio_new
Parameters:
audioPointer to the VisAudio which needs to be initialized.
Returns:
VISUAL_OK on success, -VISUAL_ERROR_AUDIO_NULL on failure.
