Typedef Documentation

typedef void(* VisLogHandlerFunc)(VisLogSeverity severity, const char *message, const VisLogSource *source, void *priv)

Functions that want to handle messages must match this signature.

  • message The message that will be shown, exactly the same as that was passed to visual_log(), but after formatting.
  • funcname The name of the function that invokes visual_log(). On non-GNU systems this will probably be NULL.
  • priv Private field to be used by the client. The library will never touch this.
