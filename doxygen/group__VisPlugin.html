typedef const VisPluginInfo*(* VisPluginGetInfoFunc)(void)

Enumeration Type Documentation

enum VisPluginFlags

Enumerate to define the plugin flags.

Plugin flags can be used to define some of the plugin it's behavior.

Enumerator:
VISUAL_PLUGIN_FLAG_NONE 

Used to set no flags.

VISUAL_PLUGIN_FLAG_NOT_REENTRANT 

Used to tell the plugin loader that this plugin is not reentrant, and can be loaded only once.

VISUAL_PLUGIN_FLAG_SPECIAL 

Used to tell the plugin loader that this plugin has special purpose, like the GdkPixbuf plugin, or a webcam plugin.
