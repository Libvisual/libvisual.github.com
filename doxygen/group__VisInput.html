typedef int(* VisPluginInputUploadFunc)(VisPluginData *plugin, VisAudio *audio)

An input plugin needs this signature for the sample upload function.

The sample upload function is used to retrieve sample information when a input is being used to retrieve the audio sample.

  • plugin Pointer to the VisPluginData instance structure.
  • audio Pointer to the VisAudio in which the new sample data is set.
Returns:
0 on succes -1 on error.
