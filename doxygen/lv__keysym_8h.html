Enumeration Type Documentation

enum VisKey

Enumerate values used within the libvisual event system for keyboard events.

The table is closely modelled after that of SDL and the SDL1.2 SDLK defines can be directly translated to those of libvisual, however some keys are left out, but these are rarely or never used.

The basic keys are also mapped as in the ASCII table so basic keyboard support is easy to implement within a libvisual client.
