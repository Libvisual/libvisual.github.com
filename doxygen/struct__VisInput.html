Detailed Description

The VisInput structure encapsulates the input plugin and provides abstract interfaces to the input.

The VisInput system provides PCM data to the visualisation elements of libvisual. This can be done through both plugins and callback functions.

Members in the structure shouldn't be accessed directly but instead it's adviced to use the methods provided.
