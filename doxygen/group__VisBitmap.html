int visual_bitmap_load ( VisVideo *  video,
const char *  filename 
)

Loads a BMP file into a VisVideo.

The buffer will be located for the VisVideo.

Keep in mind that you need to free the palette by hand.

Parameters:
videoDestination video where the bitmap should be loaded in.
filenameThe filename of the bitmap to be loaded.
Returns:
VISUAL_OK on success, -VISUAL_ERROR_VIDEO_NULL, -VISUAL_ERROR_BMP_NOT_FOUND, -VISUAL_ERROR_BMP_NO_BMP, -VISUAL_ERROR_BMP_NOT_SUPPORTED or -VISUAL_ERROR_BMP_CORRUPTED on failure.
